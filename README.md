# Animal Haven Rescue – Final Project

## Project Overview

This project is a responsive website created for Animal Haven Rescue, an animal shelter focused on helping rescued animals find safe and loving homes.

The goal of this project was to translate a UX design created in Figma into a fully functional website using Bootstrap. The website includes three main pages: Home, About, and Contact, each designed to provide a clear and user-friendly experience.

---

## My Process

I started this project by using my previous UX design work as a foundation. This included my wireframes, layout structure, and visual style created in Figma.

From there, I focused on translating that design into code by:
 Building the layout using Bootstrap’s grid system
 Recreating the structure of each page based on my wireframes
 Applying consistent colors and styling using custom CSS
 Making sure the site worked well across different screen sizes

---

## Challenges & Solutions

One of the main challenges was making the design responsive across mobile, tablet, and desktop. I solved this by using Bootstrap’s grid system and testing different screen sizes.

Another challenge was keeping the design consistent with my Figma prototype while working within Bootstrap’s structure. I adjusted spacing and layout carefully to stay as close as possible to the original design.

I also focused on accessibility by adding:
 alt text for images
 labels for form inputs
 a skip link for keyboard navigation

---

## What I Learned

Through this project, I learned how to:
 Turn a UX design into a real website
Use Bootstrap effectively for layout and responsiveness
 Improve accessibility in web design
 Organize files and structure a project properly

This project helped me better understand the connection between design and development.
